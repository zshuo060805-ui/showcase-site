道路交通事故现场安全防护典型场景图示整理

分为高速公路 / 城市快速路、普通公路 / 一般城市道路两大类，共 12 个典型场景，核心设置规则如下：
一、高速公路与城市快速路（5 类场景）
1. 一般路段・边侧车道
场景 事故位于高快速路一般路段的左侧 / 右侧边侧车道内

设置要点：
  • 预警区：沿来车方向由上游向下游布设事故管理、限速、线形诱导标志，放置于同侧道路边缘，限速值阶梯式降低
  • 警戒区：配备车辆闯入报警设备、反光锥筒，起点在道路边缘，配套警示标志；警车斜向停放作为防冲撞掩体
  • 警戒岗：至少 1 名人员在警戒区内面向来车方向值守观察
2. 一般路段・中间车道
场景 双向 6 车道及以上高快速路一般路段，事故位于中间车道

设置要点：
  • 预警区、警戒区基础设置规则与边侧车道一致
  • 必须封闭道路中心线或路侧至中间车道的范围，禁止仅封闭中间单车道
3. 隧道 / 桥梁路段
场景 事故位于高快速路隧道、桥梁路段

设置要点：
  • 事故在隧道入口 / 桥梁起点：按一般路段规则设置，防护距离适当延长
  • 事故在隧道 / 桥梁中段、出口：预警区起点提前至隧道入口 / 桥梁起点之前
  • 可视情采取限制通行或临时中断交通措施
4. 匝道路段
场景 事故位于高快速路匝道路段

设置要点：
  • 预警区起点设在进入匝道前的主线路段，其余预警设置同侧车道规则
  • 警戒区设置同侧车道规则
  • 具备通行条件可留临时通道压降车速；不具备则限流 / 断流，上游同步提示分流
5. 高速公路收费广场
场景 事故位于高速公路收费广场内

设置要点：
  • 沿收费岛出口向下游方向设置预警区，标志牌放置于事故同侧道路边缘
  • 警戒区设置同中间车道规则
二、普通公路与一般城市道路（7 类场景）
1. 直线路段・右侧车道
场景 直线路段，事故位于右侧车道内

设置要点：
  • 预警区、警戒区基础设置规则同高快速路边侧车道
  • 若事故占用非机动车道、人行道，需增设防护设施隔离非机动车与行人
2. 直线路段・左侧车道（未跨分隔线）
场景 无中央物理隔离的直线路段，事故位于左侧车道、未跨越中央分隔线

设置要点：
  • 本侧来车方向设预警区，同时对向来车方向布设事故管理、限速标志预警，标志牌置于中央分隔线
  • 本侧设警戒区，中央分隔线视情摆放锥筒隔离
  • 若道路有中央物理隔离设施，按右侧车道规则设置
3. 直线路段・跨越中央分隔线
场景 无中央物理隔离的直线路段，事故跨中央分隔线

设置要点：
  • 双向来车方向均设置预警区，标志牌置于中央分隔线
  • 双向来车方向均设置警戒区，标志牌置于中央分隔线
4. 急弯路段
场景 事故位于急弯路段

设置要点：
  • 预警区起点提前至进入弯道之前，其余按对应直线路段规则执行
  • 警戒区按对应直线路段规则执行
5. 陡坡路段
场景 事故位于陡坡路段

设置要点：
  • 预警区起点设在坡顶附近，其余按对应直线路段规则执行
  • 警戒区按对应直线路段规则执行
6. 平面交叉口・入口段
场景 事故位于平面交叉口入口段，需临时封闭

设置要点：
  • 沿来车方向由上游向下游，布设事故管理、禁止驶入标志设预警区，标志牌置于同侧道路边缘
  • 警戒区前端横向放置车辆闯入报警设备，配套警示标志；其余同直线路段右侧车道警戒规则
  • 上游同步设置信息提示与分流措施
7. 平面交叉口・出口段
场景 事故位于平面交叉口出口段，需临时封闭

设置要点：
  • 在所有可能驶入事故现场的入口路段，布设事故管理、禁止驶入、禁止转弯标志设预警区
  • 警戒区设置同入口段规则
